# TPMP Lab2 Task3

## Overview
Программа для решения двух задач:
1. Поиск чисел-близнецов (простые числа, отличающиеся на 2)
2. Работа со структурой STUD (студенты, их оценки и статистика)

## Usage
1. Сборка проекта: `make`
2. Запуск программы: `make run` или `./program`
3. Очистка: `make clean`

Программа имеет меню с тремя опциями:
- Опция 1: Поиск чисел-близнецов до заданного предела
- Опция 2: Работа с данными студентов
- Опция 3: Выход

## Additional Notes
- Структура STUD содержит поля: Name, GROUP, SES (массив из 4 оценок)
- Программа сортирует студентов по алфавиту и выводит тех, у кого средний балл > 4.2
- Исходная позиция студента сохраняется
